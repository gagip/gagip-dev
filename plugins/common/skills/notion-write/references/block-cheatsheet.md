# Notion 블록 문법 치트시트 (가독성 용도 주석)

Notion-flavored Markdown 블록을 **가독성 관점에서 언제 쓰는지**와 함께 정리한다. 완전·최신 문법은 항상 MCP 리소스 `notion://docs/enhanced-markdown-spec`을 resource-reading 인터페이스로 읽어 확인한다(이 파일은 그 위에 얹는 요약이다). **들여쓰기는 탭**을 쓴다.

---

## 진입부 블록

### 콜아웃 — 튀어야 하는 정보 (요약·경고·팁·정의)
```
<callout icon="📋" color="gray_bg">
	**이 문서**: 한 줄 요약. **결론**: 핵심 결론 먼저.
	여러 블록·자식도 가능(각 자식은 탭 들여쓰기).
</callout>
```
- 색을 **의미에 고정**: `gray_bg`=정의/요약, `yellow_bg`=팁·주의, `red_bg`=경고, `blue_bg`=정보/노트.
- 콜아웃 안 서식은 HTML이 아니라 Notion-flavored Markdown(`**볼드**`)을 쓴다.
- 남용 금지 — 모든 문단을 콜아웃으로 감싸면 강조가 사라진다.

### 목차 — 긴 문서 내비게이션
```
<table_of_contents/>
```
- 헤딩 텍스트를 자동 반영. 헤딩 위계만 잘 잡으면 목차가 공짜로 생긴다. 헤딩 5개 이상이면 상단에 배치.

## 구조 블록

### 헤딩 — H1~H3만, 건너뛰지 않기
```
# 대분류
## 섹션
### 하위
```
- H4까지 렌더되나 H5·H6은 H4로 강등된다. 실질적으로 **3단계 이내**로 유지.
- 헤딩은 첫 2단어에 핵심어를 배치(F-패턴 — 뒤 단어는 안 읽힘).

### 구분선 — 섹션 경계
```
---
```

### 토글 — 길거나 선택적인 심화를 접기
```
<details color?="Color">
<summary>펼치기 전에 보이는 라벨</summary>
	접히는 상세 내용(반드시 탭 들여쓰기)
</details>
```
또는 토글 헤딩:
```
## 섹션 제목 {toggle="true"}
	접히는 내용(탭 들여쓰기)
```
- **자식을 탭으로 들여쓰지 않으면 토글 안에 들어가지 않는다.** 레거시·FAQ·긴 절차·구현 상세를 접어 개요를 지킬 때.

## 나란히·표

### 컬럼 — 좁은 콘텐츠만 좌우로
```
<columns>
	<column>
		좌측(목차·짧은 링크 그룹·카드)
	</column>
	<column>
		우측
	</column>
</columns>
```
- 긴 본문·순서 의존 콘텐츠 금지 — 모바일에선 컬럼이 세로로 풀린다.

### 표 — 병렬 비교 (셀은 rich text만)
```
<table header-row="true">
	<tr>
		<td>항목</td>
		<td>값</td>
	</tr>
	<tr>
		<td>재시도</td>
		<td>지수 백오프 3회</td>
	</tr>
</table>
```
- 셀 안엔 헤딩·리스트·이미지 불가, `**볼드**` 같은 rich text만. **표를 스크린샷 이미지로 넣지 말 것.**
- 셀 병합은 이 포맷으로 못 한다 — 필요하면 사용자에게 노션 UI에서 하도록 안내.

## 인라인·강조

- 볼드 `**텍스트**` · 이탤릭 `*텍스트*` · 취소선 `~~텍스트~~` · 밑줄 `<span underline="true">텍스트</span>`
- 인라인 코드 `` `code` `` (개행 금지 — 여러 줄은 `` `줄1<br>줄2` ``)
- 인라인 색: `<span color="red">텍스트</span>` (배경은 `red_bg` 등)
- 블록 색: 블록 첫 줄에 `{color="Color"}`
- 링크 `[텍스트](URL)` · 인용 `> 텍스트` · 체크박스 `- [ ]` / `- [x]`
- 멘션: `<mention-page url="{{URL}}">제목</mention-page>`, `<mention-user url="{{URL}}"/>`, `<mention-date start="YYYY-MM-DD"/>`
  - **주의**: `<page url>`은 하위 페이지를 **이동**시킨다(태그 제거 시 자식 페이지 삭제). 단순 참조는 `<mention-page>`를 쓴다.

## 색 팔레트
- 텍스트색: `gray brown orange yellow green blue purple pink red`
- 배경색: 위에 `_bg` 접미사 (`gray_bg` …)

## 코드블록 주의
```` ```language ... ``` ````
- 코드블록 **안에서는 이스케이프하지 않는다** — 문자 그대로 쓴다. (백슬래시 이스케이프는 코드블록 바깥에만.)
- 다이어그램은 ` ```mermaid `. 노드 라벨에 괄호 등 특수문자가 있으면 `A["텍스트 (부연)"]`처럼 큰따옴표로 감싼다.
- **본문 전체를 코드펜스로 감싸지 말 것** — 네이티브 블록이 다 죽는다.
