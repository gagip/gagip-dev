# mobile

모바일 제품 개발(Android/iOS/Tauri) 전반을 아우르는 플러그인.
코드 리뷰·테스트 작성부터 출시 품질 검토·아키텍처 가이드라인까지 하나의 플러그인에서 다룬다.

## 스킬

| 스킬 | 설명 | 트리거 예시 |
| ---- | ---- | ----------- |
| `android-review` | Android 코드베이스의 가독성·정확성·보안·아키텍처·테스트 리뷰 | "리뷰해줘", "코드 봐줘", "PR 리뷰" |
| `android-test` | 대상 코드 분석 → 시나리오 논의 → 계획 → 테스트 코드 생성 | "테스트 써줘", "테스트 만들어줘", "단위 테스트 추가" |
| `tauri-guideline` | Tauri v2 3-레이어(Native↔Rust↔React) 아키텍처·디버깅·테스트·플러그인 선택 가이드라인 | "tauri 구조", "레이어 분리", "tauri 플러그인 골라줘" |
| `review-quality` | 플랫폼 공식 품질 기준(Android Core App Quality + Apple ARG/HIG) 기반 출시 전 정적 검토 | "출시 전 품질 점검", "스토어 심사 전 확인", "권한/네트워크 보안 점검" |

## 출처

- Android Core App Quality — https://developer.android.com/docs/quality-guidelines/core-app-quality
- Apple App Review Guidelines — https://developer.apple.com/app-store/review/guidelines/
- Apple Human Interface Guidelines — https://developer.apple.com/design/human-interface-guidelines/
