# android

Android 개발에 특화된 코드 리뷰 및 테스트 도구.

## 스킬

| 스킬 | 트리거 예시 |
| ---- | ----------- |
| `review-code` | "리뷰해줘", "코드 봐줘", "코드 점검" |
| `write-test` | "테스트 써줘", "단위 테스트 추가" |

### `review-code`

Kotlin/Compose 컨벤션 기반 코드 리뷰. 가독성, 정확성, 보안, 아키텍처, 테스트 항목을 체크한다.

### `write-test`

대상 코드를 분석하고 테스트 시나리오를 논의한 뒤 테스트 코드를 생성한다. BehaviorSpec, Fake 우선 전략, StateFlow 검증 패턴을 적용.
